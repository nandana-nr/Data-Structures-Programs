# Data Structures Programs (C)

This repository contains implementations of fundamental data structures and algorithms using the C programming language. These programs were developed as part of academic coursework to strengthen problem-solving skills and core computer science concepts.

---

## 📌 Topics Covered

### 🔹 Searching Algorithms
- `linear.c` → Linear Search  
- `binary.c` → Binary Search  

### 🔹 Sorting Algorithms
- `bubble.c` → Bubble Sort  
- `selection.c` → Selection Sort  
- `insertion.c` → Insertion Sort  
- `merge.c` → Merge Sort  
- `quick.c` → Quick Sort  

### 🔹 Data Structures
- `stack.c` → Stack Implementation  
- `queue.c` → Queue Implementation  
- `circular.c` → Circular Queue  
- `doubleended.c` → Double Ended Queue (Deque)  
- `priority.c` → Priority Queue  
- `singly.c` → Singly Linked List  

### 🔹 Trees & Advanced Structures
- `btree.c` → Binary Tree  
- `heap.c` → Heap Implementation  

### 🔹 Expression & Other Applications
- `infix.c` → Infix Expression Evaluation  
- `polynomial.c` → Polynomial Representation  
- `polysingly.c` → Polynomial using Linked List  
- `sparse.c` → Sparse Matrix Representation  
- `bfs.c` → Breadth First Search (Graph Traversal)  

### 🔹 Memory Management Simulation
- `memory_garbage_collection.c` → Simulates memory allocation, deallocation, and garbage collection using linked lists  

---
